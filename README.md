# start-blacklist
明星名字合集，以指定的分隔符输出

## Run env
python >= 3.6

## Run
```
Usage: python get_name_list.py [options]
Options:
    -f      The path to the file that contains the name(or everything you want), default 'starts.txt'
    -F      The separator between the output names, default '\r\n'
    -h      Print this and exit
```
